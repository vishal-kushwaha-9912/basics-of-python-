# Basics of Python

Welcome to the **Basics of Python** repository.
This project is created for beginners who want to learn Python programming from scratch.

## About the Project

This repository contains:

* Basic Python programs
* Practice questions
* Beginner-friendly examples
* Simple logic-building exercises
* Notes for understanding Python concepts

The goal of this project is to improve programming skills step by step while learning the fundamentals of Python.

## Topics Covered

* Variables and Data Types
* Input and Output
* Conditional Statements
* Loops
* Functions
* Strings
* Lists
* Dictionaries
* Basic Problem Solving

## Who Is This For?

This repository is helpful for:

* Beginners learning Python
* Students preparing for coding interviews
* Anyone starting programming for the first time

## How to Run the Programs

1. Install Python
2. Open the project in VS Code
3. Run any file using:

```bash
python filename.py
```

Example:

```bash
python first.py
```
